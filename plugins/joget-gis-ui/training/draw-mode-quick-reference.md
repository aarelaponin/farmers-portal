# Draw Mode — Quick Reference Card

**For office workers drawing boundaries on a desktop computer**

---

## Steps

### 1. Open the Form

Open the form that has the map. You will see the map area.

### 2. Click "Capture Boundary"

Click the **Capture Boundary** button to begin.

### 3. Choose "Draw on Map"

If you see two options, click **Draw on Map**.

### 4. Find the Right Location

Use any of these to navigate the map:

- **Search** — Click the search icon (top-left), type a place name, and select a result
- **Zoom** — Scroll your mouse wheel, or click + / - buttons
- **Pan** — Click and drag the map
- **Layers** — Click the Layers button (bottom-left) to switch to Satellite view

### 5. Click to Add Corners

Click on the map where each corner of the boundary is. A numbered marker appears for each corner.

- A dotted line follows your mouse showing where the next edge will go
- The bottom panel shows area and perimeter as you draw

> You need **at least 3 corners** to make a boundary.

### 6. Close the Boundary

When you have all your corners, close the boundary using any of these:

- **Click on the first corner** (it turns green and shows "Click to close")
- **Double-click** on the map
- **Click the "Complete" button** in the toolbar

### 7. Review

The boundary is now shown as a blue shape. Check the bottom panel:

- **Area** — size in hectares
- **Perimeter** — distance around in meters
- **Corners** — number of corners

### 8. Edit if Needed

- **Delete a corner** — Click a corner (it turns red), then press the **Delete** key
- **Add a corner** — Drag a small diamond shape on any edge to create a new corner
- **Start over** — Click the **Redraw** button

### 9. Save

If the boundary looks correct, save the form.

---

## Toolbar Buttons

| Button | What It Does |
|--------|--------------|
| **Undo** | Removes the last corner you added |
| **Clear** | Removes all corners — start fresh |
| **Complete** | Finishes the boundary (closes the shape) |
| **Edit** | Go back to editing mode after preview |
| **Redraw** | Delete everything and start over |
| **Layers** | Switch between Street, Satellite, Hybrid, and Terrain maps |

---

## Keyboard Shortcuts

| Key | What It Does |
|-----|--------------|
| Ctrl+Z (or Cmd+Z on Mac) | Undo last corner |
| Delete or Backspace | Remove selected corner |
| Escape | Finish drawing |
| + / - | Zoom in / out |
| Arrow keys | Move the map |

---

## Common Problems

| Problem | What to Do |
|---------|------------|
| "Boundary lines are crossing" (red lines) | Some edges cross each other. Click on corners near the crossing and press **Delete**, or click **Redraw** to start over. |
| Overlap warning appears | Your boundary touches another boundary. Click **Adjust Boundary** to fix, or **Save Anyway** if it is correct. |
| Area too small / too large | Add or move corners to make the boundary bigger or smaller. |
| Cannot see the location | Switch to **Satellite** view using the Layers button. Use **Search** to find the area. |

---

## Tips

- Switch to **Satellite** view to see the actual land and buildings
- Place corners where the boundary **changes direction**
- For straight edges, you only need a corner at **each end**
- For curved edges, use more corners to follow the curve
- Zoom in close for better accuracy
- Use **Undo** if you place a corner in the wrong spot
